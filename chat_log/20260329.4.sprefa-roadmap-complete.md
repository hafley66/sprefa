## Key Insights
- insta snapshot testing has friction: INSTA_UPDATE=always sometimes writes .snap.new instead of overwriting .snap in place. Need to manually mv .snap.new to .snap. cargo test across all crates can cause stale snapshot files from parallel test binary compilation.
- Rust full rebuilds triggered by background task conflicts on artifact directory lock -- 4+ minute rebuilds of entire dep tree including libsqlite3-sys C compilation
- User exploring Prolog/Datalog as future query language over match_links graph. LinkPredicate enum is a restricted term language, compile() is the translation. Real Datalog would let variable sharing compile to JOINs and equality conditions.
- CodeQL is closest prior art (Datalog over relational code database). Key differentiators: sprefa has user-authored JSON extraction rules (not compiled extractors), cross-repo linking first-class, incremental updates.
- Semgrep Pro overlap is real for dep crawling but lacks the programmable link rules and persistent cross-repo graph.
