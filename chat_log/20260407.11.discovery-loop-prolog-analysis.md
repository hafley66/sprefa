2. Explained discovery loop with examples:

   - Diamond chain: 4-round discovery (deploy → app → lib-core → lib-utils)

3. Prolog integration analysis:
   - Current scoping: lexical blocks + cross-rule refs (one-way binding)
   - Missing: unification, backtracking, bidirectional variables, recursion
   - Recommendation: Two-layer architecture (sprefa extraction → Datalog query)
   - Per-rule tables already relational - perfect interface for query layer
