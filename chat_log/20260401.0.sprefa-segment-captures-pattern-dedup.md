### Part 2: Segment captures + dedup
Created `pattern.rs` as the single source of truth for pattern matching:

```rust
enum PatternMatcher {
    Glob(GlobMatcher),
    Regex(Regex),
    SegmentCapture(Vec<Segment>),
}

enum Segment {
    Literal(String),
    Capture(String),      // $NAME
    MultiCapture(String), // $$$NAME
    Wild,                 // $_
    MultiWild,            // $$$_
}
```

Detection rule: `re:` prefix -> Regex, contains `$` -> SegmentCapture, otherwise -> Glob.

Killed 4 duplicate pattern implementations:
- git_match.rs local PatternMatcher (replaced with import)
- file_match.rs local PatternMatcher (replaced with import)
- walk.rs pipe_glob_matches (recompiled per call, replaced with pattern_matches)
- walk.rs CompiledPipeGlob (dead code, deleted)

Added `matches_with_captures()` to both CompiledGitSelector and CompiledFileSelector.
Updated `resolve_context_captures` in extractor.rs to merge pattern-level captures from segment/regex patterns alongside slot-level whole-value captures.
